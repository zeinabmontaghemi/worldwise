# Overview
This project is a single-page application built with React + Vite. Designed to track your travel adventures.It features a world map where users can click on a location to  view the city name and corresponding country flag. The site allows you to write notes for the cities youâ€™ve visited, set the date of your visit, and save these details in a city list.<br/>
